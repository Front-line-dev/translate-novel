# Web Novel translator

Translate Jp novels to Kr using LLM.

## Steps

- Set Novel from URL
  - Get Novel text from chapters
    - Translate text with LLM
    - Write translate note for next chapters

## Requirement

- Google API
- Syosetu URL
- Node.js 22
